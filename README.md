# 📊 Tableau Data Visualisation Project
### Data Technician Bootcamp – Week 2  

---

## 🧾 Overview  
This project was completed as part of a **Data Technician Bootcamp**, focusing on developing **interactive dashboards** and delivering insights through **Tableau**.

The work involved analysing **retail, sales, and analytical datasets** to identify trends, patterns, and relationships. The final dashboards are designed to support **data storytelling**, enabling users to explore insights through intuitive and interactive visualisations.
